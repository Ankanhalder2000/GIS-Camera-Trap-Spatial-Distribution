# Camera Trap Spatial Distribution Mapping (QGIS)

## Project Overview
This project presents a GIS-based spatial visualization of camera trap observations.
Using cleaned point data, the project maps the spatial distribution of wildlife detections
to understand clustering patterns and spatial arrangement of camera trap records.

The project is designed as a continuation of a Python-based data cleaning and analysis workflow.

## Objectives
- Visualize camera trap point locations in a GIS environment
- Assess spatial clustering and distribution patterns
- Produce a clean map layout
- Prepare data for further spatial analysis in future GIS projects
